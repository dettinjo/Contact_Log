# Kontakttagebuch

Eine native iOS-Anwendung zur Verwaltung persönlicher Kontakte und zur Nachverfolgung von Interaktionen. Entwickelt mit Swift und Core Data für persistente, offline-fähige Datenspeicherung.

### Funktionen

- **Kontaktverwaltung** mit Name, Telefon, E-Mail und Notizen
- **Interaktionsverlauf** zur Dokumentation vergangener Gespräche und Treffen
- **Core Data** für persistente lokale Datenspeicherung
- **Suchfunktion** zum schnellen Auffinden von Kontakten
- **Native iOS UI** mit UIKit und Auto Layout

### Hintergrund

Das Projekt entstand als Übung zur iOS-Entwicklung mit Swift, mit besonderem Fokus auf Core Data als Persistenz-Framework und die Integration nativer iOS-Designmuster.
